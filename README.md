# 🐹 地鼠潜行 - Mole Sneak

一款角色反转的网格潜行策略游戏，玩家扮演地鼠在躲避农夫锤击的同时，策略性地偷取胡萝卜。

## 🎮 游戏简介

传统的"打地鼠"游戏中，玩家扮演农夫用锤子击打地鼠。而在这款游戏中，角色完全反转：

- **你**：扮演可爱的地鼠 🐹
- **目标**：在农田里偷取尽可能多的胡萝卜 🥕
- **威胁**：躲避农夫的锤击 ⚠️
- **关卡**：3个难度递进的关卡

## 📋 游戏特色

### 第一关：基础玩法
- 10x10 网格地图
- 键盘方向键或 WASD 控制移动
- 随机生成胡萝卜，收集增加分数
- 每收集1个胡萝卜得10分
- 农夫随机在网格上显示红色预警 ⚠️，短暂延迟后落锤 💥
- 被锤子砸中则游戏结束
- 达到50分即可过关

### 第二关：智能农夫
- 12x12 网格地图
- 农夫不再完全随机，而是在网格边缘巡逻 👨‍🌾
- 新增地形障碍物：无法通行的石头 🪨
- 地鼠连续移动3次或吃胡萝卜会产生声波范围
- 声波会触发农夫警觉，朝声源位置砸锤
- 达到100分即可过关

### 第三关：技能与新威胁
- 12x12 网格地图
- 地鼠主动技能：
  - ⚡ **极速冲刺**：消耗3个胡萝卜，向指定方向移动3格
  - 🕳️ **原地遁地**：消耗5个胡萝卜，无敌3秒（可躲避所有攻击）
- 新威胁：
  - 🐕 **农场狗**：在特定路线上规律巡逻，碰到即死亡
  - 🪤 **捕兽夹**：踩中会减速，持续5秒
- 达到200分即可过关

## 📦 安装与运行

### 环境要求
- Python 3.8 或更高版本
- pip 包管理器

### 安装依赖

```bash
cd /home/ubuntu/Mole-Sneak
pip install -r requirements.txt
```

### 启动服务器

```bash
# 开发模式（端口2233）
uvicorn main:app --host 0.0.0.0 --port 2233 --reload

# 生产模式
uvicorn main:app --host 0.0.0.0 --port 2233
```

### 访问游戏

启动服务器后，在浏览器中访问：
```
http://localhost:2233
```

或部署在服务器上则访问：
```
http://<服务器IP>:2233
```

## 🎯 操作说明

### 键盘控制
- **↑ / W**：向上移动
- **↓ / S**：向下移动
- **← / A**：向左移动
- **→ / D**：向右移动
- **B**：使用遁地技能（第三关）

### 技能使用
1. **极速冲刺**：
   - 点击「⚡ 极速冲刺」按钮或按 D 键
   - 选择方向（方向键或点击方向按钮）
   - 消耗3个胡萝卜

2. **原地遁地**：
   - 点击「🕳️ 原地遁地」按钮或按 B 键
   - 消耗5个胡萝卜
   - 无敌3秒

## 🔌 API 接口

### 游戏接口

#### 新建游戏
```
POST /api/game/new?level={1|2|3}
```
创建新游戏，返回游戏状态

#### 移动地鼠
```
POST /api/game/move
Content-Type: application/json
{
    "game_id": "游戏ID",
    "direction": "up|down|left|right"
}
```

#### 使用极速冲刺
```
POST /api/game/dash
Content-Type: application/json
{
    "game_id": "游戏ID",
    "direction": "up|down|left|right"
}
```

#### 使用原地遁地
```
POST /api/game/burrow
Content-Type: application/json
{
    "game_id": "游戏ID"
}
```

#### 重新开始
```
POST /api/game/restart?game_id={game_id}
```

#### 下一关
```
POST /api/game/next-level?game_id={game_id}
```

#### 获取游戏状态
```
GET /api/game/{game_id}
```

## ⚠️ 游戏策略

### 第一关
- 观察预警格子会闪烁红色警告，尽量远离
- 快速收集胡萝卜，注意观察周围预警位置
- 不要在一个地方停留太久

### 第二关
- 注意观察农夫的巡逻位置
- 避免连续快速移动，会产生声波
- 吃胡萝卜会产生更大范围的声波
- 利用石头作为掩护

### 第三关
- 合理使用技能：
  - 冲刺可以快速逃离危险区域
  - 遁地可以躲避即将落下的锤子
- 注意农场狗的巡逻路线
- 小心捕兽夹，踩到会减速
- 技能有冷却时间，合理规划使用

## 🐛 常见问题

**Q: 游戏无法启动？**
- 检查 Python 版本是否为 3.8+
- 确认所有依赖是否正确安装

**Q: 端口被占用？**
- 修改启动命令中的端口号：
  ```bash
  uvicorn main:app --host 0.0.0.0 --port <其他端口>
  ```

**Q: 键盘操作没反应？**
- 确保点击了游戏界面获得焦点
- 检查输入法是否为英文输入法
